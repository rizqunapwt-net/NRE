## 📋 DECISION FRAMEWORK

### For Each Feature Addition, Ask:
1. **User Need?** Does a paying customer need this?
2. **Agent Fit?** Which agent should own this?
3. **Architecture?** Does it fit our design?
4. **Effort?** Can we implement in 1-4 weeks?
5. **Impact?** Will it move key metrics?
6. **Dependencies?** What else is blocked?

### Red Flags to Avoid
- ❌ Feature bloat (stick to focus)
- ❌ Premature optimization (wait for data)
- ❌ Over-engineering (KISS principle)
- ❌ Scope creep (say no more often)
- ❌ Ignoring users (listen first)
